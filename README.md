# 🏏 IPL Data Analysis Project

## 📌 Overview

This project focuses on analyzing Indian Premier League (IPL) cricket data to extract meaningful insights using Python and data visualization libraries.

The dataset contains ball-by-ball match data, allowing us to explore player performance, team statistics, and match trends.

---

## 📂 Dataset

The dataset used in this project includes:

* `matches.csv` → Match-level data (teams, winner, venue, etc.)
* `deliveries.csv` → Ball-by-ball data (runs, batsman, bowler, extras, etc.)

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas 📊
* NumPy 🔢
* Matplotlib 📉
* Seaborn 🎨
* Jupyter Notebook 📒

## 🔍 Key Analysis Performed

### 📈 Batting Analysis

* Top run scorers
* Strike rate of batsmen
* Most consistent players

### 🎯 Bowling Analysis

* Wicket-taking bowlers
* Economy rates
* Bowling performance by overs

### 🏏 Match Insights

* Toss impact on match results

### 📊 Data Visualization

* Bar plots
* Scatter plots

## 🚀 How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/ipl-data-analysis.git
```

2. Navigate to the project folder:

```
cd ipl-data-analysis
```

3. Install dependencies:

```
pip install pandas numpy matplotlib seaborn
```

4. Open Jupyter Notebook:

```
jupyter notebook
```

5. Run `analysis.ipynb`
